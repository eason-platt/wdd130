In class Exercises

Throughout the semester we have been assigned multiple activities designed to be completed within sixty minutes and with minimal preparation time. They challenged many different facets of website development at this indroductory level, and my completed assignments are linked here.

  • Apostle Spotlight
    • This in-class exercise had us demonstrate the organization of multiple images on a web page. I learned how to use a grid to keep everything aligned.
  • Prophet Card
    • Containing multiple different types of html elements inside of a division was the main objective of this assignment. It tought me the basics of CSS styling as well.
  • Devotional Table
    • For this assignment I created a table and the skeleton of a response form to add more content to it. I learned how to correctly structure the table and style it to my liking with new CSS properties.
  • Father's Day
    • The Father's Day in-class exercise involved styling a holiday website completely to our own standard. I chose to use a gradient background behind the main image and learned how to lay text boxes on top of the image while keeping them readable.
  • Grid Flags
    • After experimenting with CSS grids in prior websites, this assignment had us create country flags using colored divisions along with grid rows and columns. I practiced visualizing grids to better utilize them as I worked through each flag.
  • Positioning
    • Continuing working with grids, the objective here was to replicate template images using a grid. Some of the instances were quite difficult but they were another useful building block in my learning path.
  • Signup
    • Introducing Javascript, this exercise gave us a look at an interactive mailing list signup form. More grids and flexboxes were used.
